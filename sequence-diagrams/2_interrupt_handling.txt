title User Interrupts AI Mid-Response

User->>STT Provider: Speak input
STT Provider->>Buffer Manager: Transcribe text in 1s chunks
Buffer Manager->>LLM Provider: Predict sentence completion
LLM Provider->>Buffer Manager: Return predicted sentence
Buffer Manager->>TTS Provider: Hold response if user continues speaking
User-->>Buffer Manager: Starts talking again (Interrupt AI)
Buffer Manager->>TTS Provider: Cancel AI response
Buffer Manager->>LLM Provider: Reprocess new input
LLM Provider->>Buffer Manager: Return updated response
Buffer Manager->>TTS Provider: Send updated AI response
TTS Provider->>User: Speak updated AI-generated response
